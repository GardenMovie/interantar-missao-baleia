# Missão Baleia

Missão Baleia is an educational game, designed for children in the early years of education.

Developed during an intership at interAntar, a project from UFABC (Universidade Federal do ABC)

## Built with
- Unity
- C#
- Custom art and assets created by team members

## How to play
- Available on Google PlayStore under the same name, published by InterAntar.
